The Bazaar

A lightning fast fullstack magic the gathering card catalouge.

Feature Overview

Below you will find a general overview of the main features of The Bazaar. If you would like to see the technical implementation you can click the "learn more" link to be trasnported to the appropriate wiki page on that projects github.

Backend and database

In order to show the O(n) effiancy of my application I would need a large amount of seed data. I developed my own solution that would seed and scale my database upto an amount of my choice (ten thousand items or less) while perserving O(n) time and space complexity.

  • Learn more

Frontend pagination

My favorite modern web applications all implement some form of inifinite scroll pagination. So when I was building The Bazaar one of my goals was to implement inifinite scroll pagination. This implementation is able to continue to work as intended upto the five thousand subsequent item limit that I can safely use with a free heroku account.

Flashy but still fast.

Part of my own personal challenge with this application was to build style that was flashy and dynamic but also performance concious. So while the bazaar features lots of animations and custom behaviour it does not hamper the performance.
